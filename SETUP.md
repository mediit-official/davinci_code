# 다빈치 코드 게임 설치 및 실행 가이드

## 필요 사항

- Node.js (v14 이상)
- npm 또는 yarn

## 설치 방법

### 1. 서버 설치

```bash
cd server
npm install
```

### 2. 클라이언트 설치

```bash
cd client
npm install
```

## 실행 방법

### 개발 환경에서 실행

**터미널 1 - 서버 실행:**
```bash
cd server
npm run dev
```
서버가 http://localhost:3001 에서 실행됩니다.

**터미널 2 - 클라이언트 실행:**
```bash
cd client
npm start
```
클라이언트가 http://localhost:3000 에서 실행됩니다.

### 프로덕션 빌드

**클라이언트 빌드:**
```bash
cd client
npm run build
```

**서버 실행:**
```bash
cd server
npm start
```

## 게임 플레이 방법

1. 웹 브라우저에서 http://localhost:3000 접속
2. 플레이어 이름 입력
3. "방 만들기" 클릭하거나 기존 방에 "참가하기"
4. 상대방이 들어오면 자동으로 게임 시작
5. 자신의 턴에 상대방의 숨겨진 카드를 클릭
6. 숫자와 색상을 맞춰보세요!
   - 맞으면: 카드 공개 + 추가 턴
   - 틀리면: 자신의 카드 하나 공개 + 턴 넘김
7. 상대방의 모든 카드를 먼저 맞추면 승리!

## 게임 규칙

- 각 플레이어는 4장의 카드를 받습니다
- 카드는 0-11 숫자, 흰색/검은색 두 가지 색
- 카드는 오름차순으로 자동 정렬됩니다
- 같은 숫자인 경우 검은색이 먼저 옵니다
- 모든 카드가 공개되면 패배

## 트러블슈팅

### 연결 오류
- 서버가 먼저 실행되어 있는지 확인하세요
- 방화벽이 3000, 3001 포트를 차단하지 않는지 확인하세요

### 게임이 시작되지 않음
- 정확히 2명의 플레이어가 방에 있어야 게임이 시작됩니다
- 페이지를 새로고침해보세요

## 환경 변수 설정 (선택사항)

클라이언트의 `.env` 파일 생성:
```
REACT_APP_SOCKET_URL=http://localhost:3001
```

서버의 `.env` 파일 생성:
```
PORT=3001
```
